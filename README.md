# 📝 Bloco de Notas Moderno

Um bloco de notas web moderno e funcional com salvamento automático em localStorage, suporte a temas escuro/claro e recursos adicionais.

## ✨ Recursos

- ✅ Salvamento automático em localStorage
- ✅ Contador de caracteres e palavras
- ✅ Tema escuro/claro com persistência
- ✅ Exportar notas como arquivo `.txt`
- ✅ Limpar todas as notas com confirmação
- ✅ Notificações visuais para ações
- ✅ Design responsivo
- ✅ Acessibilidade (ARIA labels)
- ✅ Código moderno e orientado a objetos

## 🚀 Uso

1. Abra `index.html` no navegador
2. Comece a digitar suas anotações
3. As notas são salvas automaticamente
4. Use os botões para limpar, exportar ou alternar tema

## 🎨 Recursos Adicionais

- **Contador**: Exibe caracteres e palavras em tempo real
- **Exportar**: Salva suas notas em um arquivo `.txt`
- **Tema**: Alterna entre modo claro e escuro
- **Notificações**: Feedback visual para cada ação

## 💻 Tecnologias

- HTML5 semântico
- CSS3 com variáveis e media queries
- JavaScript ES6+ com classe `NoteManager`
- localStorage API
- Blob API para exportação

## 📱 Responsividade

Otimizado para desktop e mobile com suporte a temas no navegador.

## 🔒 Armazenamento

Todas as notas são armazenadas localmente no navegador (localStorage) - nenhum dado é enviado para servidores.
